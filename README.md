# EnfócatePe - Acceptance Tests

Este repositorio contiene los archivos de pruebas de aceptación del proyecto EnfócatePe, desarrollado para el curso de Ingeniería de Requisitos.

## Estructura del proyecto

### Core Business Features
- control_distracciones.feature
- pomodoro.feature

### No funcionales
- login.feature
- rendimiento.feature
- usabilidad.feature
- disponibilidad.feature

## Descripción

Los archivos están escritos en lenguaje Gherkin utilizando Given / When / Then para describir los escenarios de uso del sistema.

## Objetivo

Validar los requisitos funcionales y no funcionales del sistema enfocado en la mejora de concentración y reducción de distracciones digitales en estudiantes.
